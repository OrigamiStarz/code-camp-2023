About

CODE Camp is an annual camp held in the summer by the Glen A. Wilson CODE Team! The CODE Team design a fun and exciting curriculum to teach the basics of web development (HTML, CSS, JS). This year, it lasts 1.5 hours a day, 4 days a week, for 3 weeks, and the CODE Team has decided to also introduce competitive programming in this year's camp. By the end of the camp, campers will be able to design their own websites using HTML, CSS, and JS, and know how to solve certain types of algorithms.

FAQ

What will I get from attending?

Campers will learn the basics of web development and be introduced to competitive programming.

Do I need any prior coding experience?

Nope. All experience levels are welcome!

Do I need to bring anything?

Just water and a laptop. You can also bring snacks to eat during break.

Who are welcome to join?

This camp is open to incoming 9th to 12th graders.
